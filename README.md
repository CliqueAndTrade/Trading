## 🚀 Déploiement Rapide

### GitHub Pages (Gratuit)

```bash
# 1. Cloner le repository
git clone https://github.com/votre-username/clickn-trade.git
cd clickn-trade

# 2. Pousser vers GitHub
git add .
git commit -m "Initial commit"
git push origin main

# 3. Activer GitHub Pages dans les Settings
# Votre site sera à: https://votre-username.github.io/clickn-trade
```

### Configuration Domaine IONOs

Voir le guide complet: [DEPLOYMENT_GUIDE.md](DEPLOYMENT_GUIDE.md)

**Résumé rapide:**
1. Achetez votre domaine sur IONOs
2. Configurez les enregistrements DNS (A ou CNAME)
3. Ajoutez le domaine dans GitHub Pages Settings
4. Attendez 15-30 minutes pour la propagation DNS

## 🔧 Personnalisation

### Changer les Couleurs

Modifiez les variables CSS au début de `style.css`:

```css
:root {
    --primary-color: #9945ff;
    --accent-green: #00ff41;
    --dark-bg: #0a0515;
    /* ... autres variables ... */
}
```

### Modifier le Contenu

- **Texte**: Modifiez directement dans `index.html`
- **Images**: Remplacez les fichiers .png
- **Vidéo**: Mettez à jour le lien vers votre vidéo

### Ajouter des Formulaires

Dans `script.js`, configurez les boutons pour rediriger vers:
- Système de paiement (Stripe, PayPal)
- Formulaire d'inscription (Typeform, etc.)
- Système de support (Discord, etc.)
